Golf Ball API

Information

What is the point of this API?

This API can be used to acquire golf ball information about a variety of golf balls.

How can I use this API?

Requests to this API can be made to the endpoint listed below. Simply add the kind of golf ball you are searching for as a parameter added to the end of the URL above

Example:

The request of https://golf-ball-api.herokuapp.com/api/prov1 will return the following JSON object:

{
"name":"prov1",
"brand":"titleist",
"color":"white",
"swingSpeed":"medium",
"hardness":"soft",
"quality":"great"
}

Endpoint:https://golf-ball-api.herokuapp.com/api/:golfballname Endpoint:https://golf-ball-api.herokuapp.com/api/:golfballname Endpoint:https://golf-ball-api.herokuapp.com/api/:golfballname Endpoint:https://golf-ball-api.herokuapp.com/api/:golfballname

Additional Info:

If the endpoint requested does not exist, a magic ball will be returned.
